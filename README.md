# Muhammad Usama — Portfolio

Interactive portfolio showcasing my work as a Full-Stack and Automation Engineer.

## Highlights

- 3D robot-guided experience
- Full-stack, WordPress and AI automation projects
- Professional profile and downloadable CV
- Responsive design with light and dark themes
- GSAP animations and interactive project gallery

## Built with

HTML, CSS, JavaScript, Three.js and GSAP.

## Run locally

Serve the project through a local HTTP server, then open `index.html` in your browser.
